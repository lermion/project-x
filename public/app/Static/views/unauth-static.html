О сервисе Place People

Информация хранится в системе в виде объектов.

Объект состоит из "заголовка", "текста" и может содержать в себе файл или несколько файлов. Так же в нем есть другая системная информация, например время создания, изменения, доступ и тому подобное...

Если в объект загружена картинка, она показывается при просмотре данного объекта и в краткой информации о объекте. Для создания объекта можно использовать кнопку "создать" на "моей страница", при этом созданный объект отобразиться еще и на самом "столе".

Так же создание доступно при просмотре объектов "написать отзыв", "добавить статью в раздел", "добавить файлы в раздел", если для этого есть необходимые права доступа.

При просмотре объекта могут быть доступны следующие действия:

  • "написать отзыв";
  • "добавить статью в раздел" - просматриваемый объект будет использоваться как раздел и созданный объект появится в нем;
  • "добавить файлы в раздел" - просматриваемый объект будет использоваться как раздел, для каждого загруженного файла будет создан объект с этим файлом, который появится в разделе;
  • "добавить из запомненных" - возможность вставлять уже созданные ранее объекты в раздел, можно вставлять любые объекты, которые доступны для просмотра, объекты выбираются из "запомненных";
  • "запомнить" - добавления текущего объекта в список "запомненных", для последующей вставки в раздел;
  • "в моё" - добавление объекта в "запомненные", на "мою страницу", в "аватары" и тому подобные действия;
  • "редактировать" - правка объекта;
  • "удалить" - полное удаление объекта и всех его связей, без возможности восстановления;
